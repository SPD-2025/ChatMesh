Arquitetura do Projeto: ChatMesh

Sistema distribuído de chat com comunicação ponto-a-ponto (P2P), replicação e sincronização de mensagens entre múltiplos peers.

Peer 1

Interface de envio de mensagens
Conecta a múltiplos peers e armazena logs

Python Socket TCP Log Local
Peer 2

Recebe mensagens e sincroniza o histórico com outros peers

Multiprocessing Sincronização Backup Local
Peer 3

Capaz de reconectar após falha e recuperar histórico

Tolerância a Falhas Reconexão Métricas

Arquitetura Técnica

Baseada em rede mesh com comunicação descentralizada

P2P Full Mesh Threads/Multiprocessing Docker Containers Rede Virtual Docker

Tecnologias Utilizadas

Python, Sockets, Multiprocessing, FastAPI (interface), Docker, Docker Compose

Suporte a escalabilidade horizontal e sincronização ponto-a-ponto entre nós
